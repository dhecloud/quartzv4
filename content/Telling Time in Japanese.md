# Telling Time {今|いま}{何|なん}{時|じ}{何分|なん|ぷん}ですか。
## {時|じ}
#### 4/7/9 uses different pronunciation
- よじ (yo not yon)
- しちじ
- くじ
## {分|ぷん}

#### 2/5/7/9 uses ふん
- {二分|にふん}
- {五分|ごふん}
- {七分|ななふん}
- {九分|きゅうふん}
#### 3/4/ uses ぷん
- {三分|さんぷん}
- {四分|よんぷん}
#### 1/6/8/10 uses っぷん
- {一分|いっぷん}
- {六分|ろっぷん}
- {八分|はっぷん}
- {十分|じゅっぷん}
